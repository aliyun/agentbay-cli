## 🔐 认证方式

使用此命令前需要完成认证，支持两种方式：

### 方式 1: OAuth 登录（推荐本地开发使用）

```bash
# 先登录
agentbay login

# 然后使用功能
agentbay <命令路径> --<参数> "<值>"
```

### 方式 2: AK/SK 环境变量（推荐脚本和 CI/CD 使用）

```bash
# 配置环境变量
export AGENTBAY_ACCESS_KEY_ID="your-access-key-id"
export AGENTBAY_ACCESS_KEY_SECRET="your-access-key-secret"

# 使用功能
agentbay <命令路径> --<参数> "<值>"
```
